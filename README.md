# Minecraft 1.2.3_01.1

**A handmade, beta-quality, impossible-to-beat creepy Minecraft clone.**  
Built with Java and LWJGL, this is not your average block game. Expect bugs, crashes, and things that should not exist.

---

## Name & Description

**Minecraft 1.2.3_01.1 (Creepy 3D)** – a procedurally generated 3D sandbox where the blocks whisper.  
Features include:
- A world that hates you.
- Inventory that mocks you.
- Chat events that hunt you.
- Herobrine, Null, and friends from places you don't remember.
- A crash screen that feels like home.

---

## Handmade

Every line of code was typed by human fingers (and a little help from DeepSeek). No AI‑generated bloat – just pure, terrifying craftsmanship.

---

## Difficulty

**Beat it – impossible**  
There is no ending. There is no victory. The only way to "win" is to crash in the most spectacular way possible. Can you survive the Null event? Can you outrun Herobrine? No. No you can't.
